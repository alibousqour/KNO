# KNO AI OS 🤖
**An Advanced, Modular AI Operating System with Long-Term Memory & Hardware Awareness**

![Python](https://img.shields.io/badge/Python-3.9%2B-blue) ![Platform](https://img.shields.io/badge/Platform-Windows%20|%20Linux-orange) ![License](https://img.shields.io/badge/License-MIT-green)

**KNO** is a sophisticated AI Operating System framework designed to go beyond simple voice assistance. It integrates a dedicated Hardware Abstraction Layer (HAL), semantic memory, and autonomous process management to create a truly interactive and self-aware digital environment.

## ✨ Key Features

* **Autonomous Process Management**: KNO monitors system resources and can independently manage or "heal" failing processes.
* **Semantic File System (SFS)**: Beyond keyword search, KNO understands the context and meaning of your files for intelligent retrieval.
* **Hardware-Integrated HAL**: Real-time interaction with system hardware including CPU metrics, RAM management, and thermal monitoring.
* **Multi-Model Intelligence**: Leverages the power of OpenAI and Gemini APIs for high-level reasoning and vision capabilities.
* **Immersive GUI**: Features a cinematic, terminal-style interface inspired by eDEX-ui for a futuristic user experience.

## 🛠️ System Requirements

* **OS**: Windows 10/11 (LTSC recommended) or Linux.
* **CPU**: Intel Core i7-4610M or equivalent.
* **RAM**: 8GB Minimum.
* **Peripherals**: USB Microphone, Camera (for Vision), and a secondary display for the eDEX dashboard.
