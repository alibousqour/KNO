## How to Use

### Run the Agent
```bash
python agent.py
```
The consent system will:
1. Load `settings.json` on startup
2. Initialize audit logging to `logs/audit.log`
3. Show approval dialogs for dangerous operations
4. Log all decisions to audit trail

### Customize Behavior
Edit `settings.json` to:
- Change autonomy level
- Allow/ask/disable specific permissions
- Configure self-evolution settings
- Adjust audit logging retention

### View Audit Trail
```bash
cat logs/audit.log
```

## Rollback (If Needed)
If you want to remove the consent system:
1. Delete `settings.json`
2. Delete `consent_manager.py`
3. Remove ConsentManager import from agent.py
4. Remove 3 `consent_manager.request_approval()` calls from SystemActionEngine

The agent will work normally without consent checks.
