     ├─ Color progression: Green → Yellow → Red

     └─ History tracked in task_manager

     ├─ Auto-export to edex_status.json every 5 seconds
     ├─ Includes system health, process metrics, healing events
     ├─ JSON-formatted for cross-system consumption
     ├─ Thread-safe serialization
     └─ Integration bridge for eDEX-UI dashboard

⚡ QUICK START (30 seconds)

  Step 1: Copy this code to your agent.py
  ──────────────────────────────────────
  from hardware.processes import ProcessRegistry, ProcessManager
  from BotGUI_new import FuturisticBotGUI
  import tkinter as tk

  # Create components
  registry = ProcessRegistry()
  manager = ProcessManager(registry)
  root = tk.Tk()
  gui = FuturisticBotGUI(root)

  # Connect monitoring (1 line!)
  gui.set_process_registry(registry)

  # Done! Monitoring is automatic
  root.mainloop()
